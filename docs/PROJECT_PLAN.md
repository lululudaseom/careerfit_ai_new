# PROJECT_++[PLAN.md](http://PLAN.md)++ — CareerFit AI 프로젝트 기획 초안



## 1. 문제 정의

취업, 공모전을 준비하며, 어떤 스킬을 준비해야 할지, 무엇부터 시작해야 할지, 다른 사람들은 얼마나 준비해서 오는지 감이 잡히지 않아 어려웠다.
CareerFit Ai가 직무별 요구 스킬을 데이터로 정리하고,내 역량과 비교해 부족한 것들을 알려준다. 어떤 지식들이 필요하며 그 지식은 어디서 습득할 수 있는지 찾아주길 바란다.


## 2. 사용자 정의

- 대상: 취업준비생, 대학생



- 사용 시나리오:

1. 사용자가 전공, 보유 스킬, 관심 직무를 입력한다.

2. CareerFit AI가 데이터를 검색해 역량 매칭 결과를 반환한다.

3. 어떤 데이터를 기반으로 답변했는지 출처를 함께 표시한다.



## 3. 데이터 계획

- 취업 공고 컬럼: 회사명, 직무명, 요구 스킬, 우대 사항, 지원 자격, 업무 내용, 마감일

- 공모전 컬럼: 공모전명, 주최, 분야, 참가 자격, 요구 역량, 상금, 마감일

- 데이터 출처: 강사 제공 목업 CSV (3일차)



## 4. 기능 목록

1. 역량 매칭 분석
2. 부족 역량 추천
3. RAG 기반 근거 답변
4. 출처 표시
